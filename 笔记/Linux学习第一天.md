# 4. 服务器领域 #
	  Linux在服务器领域的应用是最强的
	  Linux免费、稳定、高效等特点在这里得到了很好地体现，尤其是在一些高端领域尤为广泛（c/c++/php/java/python/go）

# 5. 嵌入式领域 #

    Linux运行稳定、对网络的良好支持性、低成本，且可以根据需要进行软件剪裁，内核最小可以达到几百万kb等特点。近些年来在嵌入式的领域应用得到非常大的提高。


    主要应用：机顶盒、数字电视、网络电话、远程交换机、手机、pda等。
